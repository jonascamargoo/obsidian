## Os tipos de testes

1. Funcional -> teste final em ambiente controlado
2. Smoke test -> momento que o deve faz o ultimo teste antes de entregar para o cliente fazer o teste funcional
3. Integração -> testa se as partes do sistemas interagem entre si adequadamente
4. Segurança -> visando possíveis ataques de segurança
5. Carga -> estressa o sistema
6. Unitário -> foca exclusivamente a unidade, em POO, um método. É um trecho de código que executa outro trecho de código e verifica se executou corretamente
