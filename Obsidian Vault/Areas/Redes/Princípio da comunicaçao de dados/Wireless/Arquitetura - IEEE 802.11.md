### Stations (Unidades Móveis – UMs)

As estações, ou unidades móveis (UMs), são dispositivos que se conectam à rede sem fio. Esses dispositivos podem ser laptops, smartphones, tablets, ou qualquer outro dispositivo com capacidade Wi-Fi. Eles são responsáveis por se comunicar com os pontos de acesso para enviar e receber dados.

### Ponto de Acesso – Access Point (AP)

O ponto de acesso (AP) é o dispositivo que atua como uma ponte entre as unidades móveis e a rede cabeada. O AP transmite e recebe sinais de rádio para se comunicar com os dispositivos sem fio e permite que esses dispositivos acessem a rede local e a Internet.

### Basic Service Set (BSS)

O Basic Service Set (BSS) é o conjunto básico de unidades móveis que se comunicam entre si dentro de uma área coberta por um ponto de acesso. Há dois tipos principais de BSS:

- **Infraestrutura BSS**: Inclui um ponto de acesso (AP) que serve como uma estação central, e todas as UMs se comunicam através desse AP.
- **Ad hoc BSS**: Não inclui um AP. As UMs se comunicam diretamente umas com as outras.

### Distribution System (DS)

O Distribution System (DS) é a infraestrutura que interconecta múltiplos BSSs. Ele permite que as estações em diferentes BSSs se comuniquem entre si. Geralmente, o DS é uma rede cabeada (como Ethernet) que conecta vários pontos de acesso.

### Extended Service Set (ESS)

O Extended Service Set (ESS) é formado por dois ou mais BSSs interconectados através de um DS. O ESS permite a mobilidade dos dispositivos entre diferentes BSSs sem perder a conexão, pois a rede é vista como uma única entidade para as UMs. Isso é crucial para garantir a continuidade da conexão em áreas maiores, como em um campus universitário ou em um grande escritório.

### Funcionamento Geral

1. **Estação (UM)**: Um dispositivo sem fio que deseja se conectar à rede.
2. **Ponto de Acesso (AP)**: O AP facilita a comunicação entre as UMs e a rede cabeada.
3. **BSS**: Conjunto de UMs que se comunicam com um único AP.
4. **DS**: Sistema que conecta múltiplos APs para formar uma rede maior.
5. **ESS**: Conjunto de BSSs interconectados, permitindo a mobilidade contínua dos dispositivos entre diferentes áreas da rede.

Essa estrutura permite que redes sem fio sejam escaláveis, flexíveis e capazes de cobrir grandes áreas, fornecendo conectividade contínua e confiável para os usuários móveis.
